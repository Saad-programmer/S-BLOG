Install Go on Raspberry Pi OS Shortest Tutorial

#raspberrypi #go #linux

This may be the shortest tutorial on how to install Go on Raspberry Pi OS.

1. Connect to your Raspberry Pi

Access your Raspberry Pi terminal using Putty / UART / connecting a monitor & keyboard directly into your Pi. Once you're in, navigate to your home directory:

cd ~/

2. Download Go for ARMv6

If you are on a different architecture of Linux, see the official download page. For Raspberry Pi OS (ARMv6), grab Go 1.15.3:

wget https://golang.org/dl/go1.15.3.linux-armv6l.tar.gz
Note: For newer Raspberry Pi models (Pi 2/3/4/5 with ARMv7 or ARM64), consider downloading the latest version from go.dev/dl/. This tutorial keeps the classic minimal version.

3. Extract to /usr/local

sudo tar -C /usr/local -xzf go*

4. Add Go to your PATH

Append the export line to your .bashrc file:

echo 'export PATH=$PATH:/usr/local/go/bin' >> .bashrc

Or manually edit using nano:

sudo nano .bashrc

Then add the following line at the bottom of the file:

export PATH=$PATH:/usr/local/go/bin

5. Reload .bashrc & verify

source ~/.bashrc
go version

✅ Done! You should see output similar to: go version go1.15.3 linux/arm. That means Go is successfully installed on your Raspberry Pi OS.

Test your Go installation

Create a classic "Hello, world!" to confirm everything works:

mkdir ~/go-test
cd ~/go-test
nano hello.go

Paste the following:

package main

import "fmt"

func main() {
    fmt.Println("Hello from Raspberry Pi & Go!")
}

Then build and run:

go build hello.go
./hello

Expected output: Hello from Raspberry Pi & Go!

Pro tip: If you prefer the latest Go release, replace the wget URL with a recent ARM version. Example for Go 1.22+ (armv6l): wget https://go.dev/dl/go1.22.0.linux-armv6l.tar.gz. Always check the official Go download page.

Need more details? The complete Go documentation is available at golang.org/doc/. Happy coding on your Raspberry Pi!
